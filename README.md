# Interview Question 08

Interview Question 08 that make for technical skill test.

## Tech stack I chose

- SQLite
- Golang
- Vue + tailwind

> The database file is located "/backend/database/questions.db"

## How to start

### Back-end

- Go to 'backend' path with this command `cd backend`
- Run Golang backend service with this command `go run main.go`
- The API service should start on `http://localhost:8080`

### Front-end

- Go to 'frontend' path with this command `cd frontend`
- Install Node dependency with this command `npm i`
- Run Vue with this command `npm run dev`
- The service should start on `http://localhost:5173`
- Visit `http://localhost:5173` in any browser to get start.

**Enjoy!**
